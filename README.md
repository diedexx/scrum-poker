## Description

Refinement tool for Scrum-teams that work remotely.
Created this to learn development techniques and provide my team with a better tool.


Uses Vue and WebSockets via NestJS.

Pull requests and issues are very welcome.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
